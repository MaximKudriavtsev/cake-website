## Обзор

Проект по созданию масштабного десертного фуршета для корпоративного мероприятия крупной IT-компании. Задача состояла в разработке 7 видов пирожных и капкейков в едином стиле, отражающем бренд заказчика, с учетом разных диетических предпочтений гостей.

## Ключевые особенности

- **Брендирование десертов**: Интеграция цветовой палитры и элементов логотипа компании в дизайн всех десертов
- **Диетические линейки**: Разделение на классическую, вегетарианскую и безглютеновую серии
- **Интерактивные элементы**: Создание десертной станции с возможностью комбинирования топпингов
- **Логистическая оптимизация**: Разработка системы доставки и сервировки для сохранения идеального вида десертов

## Технологии и методы

- **Модульное производство**: Система сборки, позволяющая масштабировать производство без потери качества
- **Стабилизация кремов**: Использование термостабильных техник для сохранения формы при комнатной температуре
- **Порционная фасовка**: Индивидуальная упаковка для каждого типа десерта с учетом его особенностей
- **Система маркировки**: Четкая идентификация десертов для гостей с диетическими ограничениями

## Сложности и решения

Главный вызов — обеспечение одинаково высокого качества 450 порций десертов и их доставка к точному времени мероприятия. Решением стало разделение производства на этапы: за 3 дня — основные элементы, за 1 день — сборка, в день мероприятия — финальный декор и транспортировка в специально разработанных контейнерах.

## Результат

Успешно реализован десертный фуршет на 320 человек. Получен повторный заказ на ежегодное корпоративное мероприятие и рекомендации для 3 компаний-партнеров. Разработанная система масштабирования производства позволила впоследствии брать заказы на мероприятия до 500 человек.
